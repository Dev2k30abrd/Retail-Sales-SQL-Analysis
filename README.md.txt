# 🛒 Retail Sales SQL Analysis

## 📌 Project Overview
This project analyzes retail sales data using SQL to generate business insights about customers, products and revenue trends.

## 🛠 Tools Used
- MySQL
- SQL

## 🗂 Database Schema
The database contains 4 tables:

- Customers
- Products
- Orders
- Order_Items

## 📊 Business Questions Solved
- Customers per city
- Top spending customers
- Customers with no orders
- Best selling products
- Revenue by category
- Monthly & daily sales trends
- Customers spending above average
- Product price analysis
- Customer segmentation (High / Medium / Low value)

## 🔍 Key SQL Concepts Used
- GROUP BY & Aggregations
- INNER JOIN & LEFT JOIN
- HAVING
- Subqueries
- Date Functions
- CASE WHEN

## 🎯 Project Outcome
Built a relational database and performed end-to-end sales analysis using SQL.

This project demonstrates SQL skills for Data Analyst roles.
